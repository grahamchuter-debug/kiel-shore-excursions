# Izmir Shore Excursions

Independent Izmir cruise planning site for passengers calling at Alsancak — gateway to Ephesus and the Aegean.

**Domain:** https://izmirshoreexcursions.com

## Development

```bash
npm install
npm run dev
```

## Build & QA

```bash
npm run download-images
npm run build
npm run check-links
npm run seo-qa
```

## Deploy

```bash
npm run pages:deploy
```
